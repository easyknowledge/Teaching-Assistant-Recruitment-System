## About The Project

This project aims to streamline the Teaching Assistant (TA) recruitment process for BUPT International School. Instead of relying on scattered forms and spreadsheets, the team is designing a lightweight system that supports job posting, applicant management, and recruitment tracking.

The repository documents our Agile workflow for EBU6304, including requirements discovery, backlog planning, prototype design, and milestone-based delivery. At the current stage, the focus is the first assessment deliverables due on **22 March 2026**.

## Key Features

* Role-based flows for applicants, module organisers, and administrators
* Product backlog with user stories, priorities, acceptance criteria, and iteration planning
* Interactive prototype for demonstrating core user journeys
* Coursework-aligned implementation plan based on Java and file-based storage

## Collaboration Workflow

To satisfy the coursework requirements and keep contribution records clear:

* each member should work on a personal branch named as `{name}/{function}`
* each change should be committed and pushed through a pull request
* every pull request should be approved by at least one teammate before merge
* all meaningful progress should be visible on GitHub before each assessment checkpoint
